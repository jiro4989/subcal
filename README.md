# subcal (subnet calc)

サブネットの計算(10進数←→2進数)とかCIDRの計算をする

## できること

- 10進数、2進数相互変換
- IPリストの一覧出力
- ネットワーク部とホスト部の色分け

## 使い方

```bash
subcal 126.227.243.233/32
# -> 01111110.11100011.11110011.11101001
```
```bash
subcal list
# dec	bin
# 0.0.0.0/0	00000000.
# 0.0.0.0/1	00000000.
# 0.0.0.0/2	00000000.
```
